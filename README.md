# Premium Interactive Apology Website

A static, responsive, cinematic apology website built with HTML, CSS, and vanilla JavaScript.

## Edit points

- Letter text: `script.js`, `letterText`
- Gallery images and captions: `script.js`, `memories`
- Music placeholder: `index.html`, `#backgroundMusic`
- Colors and visual tone: `styles.css`, `:root`

## Deploy on Render

Create a new Static Site on Render and connect the GitHub repository. The included `render.yaml` publishes the project root.
